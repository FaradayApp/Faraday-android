Faraday - это клиент для протокола Matrix, настроенный на работу преимущественно с пользовательскими серверами.
По сравнению с другими клиентами, он позволяет более гибко работать с функциональностью Matrix, а также работать с командой, не опасаясь утечек информации

Протокол Matrix - это децентрализованный способ обмена сообщениями со сквозным шифрованием, позволяющий совместно использовать историю сообщений на нескольких устройствах и многое другое, возможное благодаря нашему клиенту

Faraday имеет множество улучшений:
- Поддержка нескольких серверов в одном приложении
- Мультиаккаунт для разных пользовательских серверов
- Уведомления для мультиаккаунта с автопереходом на сервер, с которого пришло уведомление
- Виджеты для совместной работы команды с интеграцией сервисов Taiga, Cryptpad
- Nuke-пароль для очистки данных приложения
- Возможность .onion подключения к серверу
- Общие улучшения для пользователя

Faraday имеет открытый код: https://github.com/FaradayApp/Faraday-android
Подробнее о протоколе Matrix: https://matrix.org/
